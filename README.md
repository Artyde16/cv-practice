# Приложение, разработанное в рамках летней практики

## Описание

Приложение разработано в Python и реализует базовый функционал для работы с изображениями:
- загрузка изображения из файла (JPEG, PNG);
- захват фотографии с веб-камеры;
- отображение изображения в окне приложения;
- отображение красного, зеленого и синего каналов изображения;
- применение негативного фильтра;
- повышение яркости изображения на введенное пользователем значение;
- рисование красного круга на изображении по координатам и радиусу, введенным пользователем.

## Версия Python

Проект написан на версии Python 3.13.14.

## Установка и запуск

1. Создайте виртуальное окружение в каталоге проекта:
   ```bash
   python -m venv venv
   ```
2. Активируйте виртуальное окружение:
   - Windows:
     ```bash
     venv\Scripts\activate
     ```
   - macOS / Linux:
     ```bash
     source venv/bin/activate
     ```
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
4. Запустите приложение:
   ```bash
   python main.py
   ```

## Примечания по работе с веб-камерой

Если приложение не может подключиться к веб-камере:
- убедитесь, что камера подключена и не используется другой программой;
- дайте приложению разрешение на доступ к камере.

## Возможные проблемы

При запуске приложения может появляться ошибка, связанная с отсутствием модуля tkinter. Это часто бывает в минимальных установках Python.

Решения по ОС:

- Windows:
   - На Windows часто проблема в том, что при установке Python не была отмечена галочка «tcl/tk and IDLE». Если Python уже установлен, можно запустить установщик в режиме «Modify» и установить этот компонент заново. 

- Linux:
    - В Linux пакетные менеджеры иногда не включают Tkinter по умолчанию. Команды для установки зависят от дистрибутива:
    - Для Ubuntu/Debian:
      ```bash
      sudo apt install python3-tk
      ```
    - Для Fedora/Red Hat:
      ```bash
      sudo dnf install python3-tkinter
      ```
- macOS:
   - При использовании системного Python tkinter обычно установлен. При использовании Homebrew-версии Python установите через brew и убедитесь, что установлены зависимости Tcl/Tk:
      ```bash
      brew install python-tk
      ```
